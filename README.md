# Screenery #

Prototype of a simple web API registry to manage desktop spaces using wmctrl.

## Running ##

    Ubuntu 14.04:
    
    $ virtualenv-3.4 -p python3 .env
    $ . .env/bin/source
    (.env)$ pip install -r requirements.txt
    (.env)$ python screenery/app.py

View at: http://localhost:5001/
